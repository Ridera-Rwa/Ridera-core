# Core Protocol Contracts

This directory contains Ridera’s core protocol smart contracts.

## Included Contracts
- `ProofRegistryWithSigner.sol` – On-chain registry for signed cycle proofs

## Purpose
- Anchor verified off-chain data on-chain
- Maintain immutable cycle history
- Enable trust-minimized reward distribution

## Notes
- Contracts are testnet-only
- Logic is subject to change
- Not audited
