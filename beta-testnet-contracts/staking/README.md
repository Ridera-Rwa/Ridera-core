# Staking Contracts (Beta)

This directory contains Ridera’s **staking and reward distribution contracts**.

## Included Contracts
- `EnhancedStakingV2.sol` – Multi-deposit staking with lock options and cycle-based rewards

## Responsibilities
- Manage user stakes
- Apply lock-based reward boosts
- Distribute rewards using APY from YieldVault
- Handle reward claiming and early unlock penalties

## Notes
- Testnet-only
- Subject to change
- Not audited
