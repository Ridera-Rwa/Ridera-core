# Ridera Protocol

Ridera is a Real-World Work (RWW) tokenization protocol that converts verified gig-worker earnings into standardized on-chain productivity units (SRU).  
It transforms global mobility and delivery work into a verifiable, transparent, and programmable financial layer built on Base.

Ridera is not a ride-hailing app.  
It is the infrastructure layer that standardizes and verifies real-world work, enabling decentralized worker identity and sustainable reward systems.
